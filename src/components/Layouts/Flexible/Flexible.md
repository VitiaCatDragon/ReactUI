# Flexible

```jsx
<>
    <Flexible direction="column">
        <button>1</button>
        <button>2</button>
        <button>3</button>
    </Flexible>

    <Flexible direction="row">
        <button>1</button>
        <button>2</button>
        <button>3</button>
    </Flexible>

    <Flexible direction="column" reverse={true}>
        <button>1</button>
        <button>2</button>
        <button>3</button>
    </Flexible>
                
    <Flexible direction="row" reverse={true}>
        <button>1</button>
        <button>2</button>
        <button>3</button>
    </Flexible>
</>
```
